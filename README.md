# Predicting Price Moves with News Sentiment

## Project Overview

**Predicting Price Moves with News Sentiment** is a Python project that combines **natural language processing (NLP)** and **financial time series analysis** to investigate how financial news sentiment influences stock price movements. Key features include:

- Sentiment analysis of news headlines.
- Correlation analysis between sentiment scores and stock price changes.
- Configurable, reproducible ML workflows using YAML configurations.
- Modular, production-ready Python package structure for scalable experimentation.

The project provides actionable insights for financial forecasting, emphasizing **reproducibility, scalability, and clarity**, making it suitable for both research and production-grade ML pipelines.
